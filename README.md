# ChineseSemanticKB
ChineseSemanticKB,chinese semantic knowledge base, 面向中文处理的12类语义常用词典，包括34万抽象语义库、34万反义语义库、43万同义语义库、停用词等，可支持句子扩展、转写等多种应用场景。
